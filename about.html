About

Radial Bitmap Typography is an experimental typeface system that transforms traditional bitmap fonts into dynamic, radial compositions.

This project explores the intersection of geometric design, computational art, and typography. Each character is constructed from a 5×5 bitmap grid, then rendered through various radial algorithms to create mesmerizing visual patterns.

The tool offers 15+ rendering modes, from basic radial lines to complex checkerboard patterns and twisted grids. Users can control parameters like radius, rotation, axis count, and create multi-layered compositions.

Features:
• Real-time preview with 12x magnifier
• Multiple layout modes (left, right, center, circular, square)
• Color customization with gradient picker
• GIF animation export
• SVG download for further editing

Technical Stack:
JavaScript, SVG, CSS3, HTML5

Created as part of the Bauhaus Typography Course, 2025.
Exploring the future of parametric typography.
